Modified [M]

This sign M will come when you are making changes in the repo but you have not added it or committed it..!

Untracked [U]

This sign "U" will appear when you add a new in the repo and git doesn't know that you have created a new file

Unmodified

This means that you have not changed anything. The file is unchanged

Staged

This means file is ready to be committed. when we add a file to the staging area, it is ready to be committed. but is not committed yet.
